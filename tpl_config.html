How to use these two files
Quick reference for admins
📊 tpl_dashboard.html
Share this file (or its hosted link) with ROMs, your team, leadership. View-only — no settings, no config. Reads changes you make here automatically.
⚙️ tpl_config.html
Keep this file private. Use it to update ROM store counts, exclude holidays, and upload new data. Changes save automatically and reflect in the dashboard.
⚠️ Both files must be opened in the same browser on the same device for live sync to work (via browser localStorage). For cloud hosting, we'll connect them via a shared data file instead.
